# Labyrinthe68k

## Bienvenue sur le repo du projet d'assembleur 68k.

Vous y trouverez un jeu de **labyrinthe** développé pour le simulateur [Easy68k](http://www.easy68k.com/) dans le cadre d'un projet du cours d'architecture des ordinateurs à l'université.

Pour **tester** le jeu, rendez-vous dans l'onglet [Releases](https://github.com/Neptune-dev/Labyrinthe68k/releases), où vous trouverez l'archive de la dernière version, contenant un fichier exécutable `.S68` par *SIM68k*, accompagné de fichiers `.L68` et `.tilesheet` nécessaires au lancement.

Veuillez trouvez ci-joint la doc : [Documentation](doc/Documentation.md)
